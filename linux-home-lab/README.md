# Linux Ubuntu Home Lab

Laboratório prático criado para desenvolver habilidades em administração Linux utilizando Ubuntu Server.

## Objetivos

- Aprender administração básica de sistemas Linux.
- Documentar comandos e procedimentos.
- Simular tarefas comuns de ambientes corporativos.
- Construir um portfólio técnico para futuras oportunidades profissionais.

# 1. Installation

- Download da ISO Ubuntu Server.
- Criação da máquina virtual.
- Configuração inicial.
- Definição de hostname.
